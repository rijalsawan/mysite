YumMomo

This is the project I made for the Business named YumMomo. The technologies that i used to make this website includes, NextJS- A full-stack React framework, MongoDb, Material Ui, Tailwind CSS.

Features: Cart: Users can add any items and their quantity to cart and can checkout easily. User Account: Every user can signup and login within the app after which user can see their orders and update their account. For admin: Admin can add products, remove products, update products, Admin can see all the orders that is ordered by user.
